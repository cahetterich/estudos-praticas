  • display_flex : identificar que a tag vai ser o container de flaxbox
  • flex-direction : identificar qual vai ser o eixo principal e perpendicular
  • justify-content : alinhar os elementos no eixo principal
  • align-items: alinhar os elementos no eixo perpendicular
  • flex-wrap: define se os itens flexíveis são forçados em uma linha ou podem ser agrupados em várias linhas
  • align-content: define a distribuição do espaço entre e ao redor dos itens de conteúdo
  • gap: define o tamanho do espaço entre as linhas e entre as colunas no flexbox, grade ou layout de múltiplas colunas
  • arow-gap: define o tamanho do intervalo (medianiz) entre as linhas de um elemento
  • column-gap: especifica o espaço entre as colunas na grade, flexbox ou layout de várias colunas
